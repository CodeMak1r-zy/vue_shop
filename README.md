# vue_shop
一个基于Vue2 和 elementUI 的电商后台管理系统
主要技术栈有:

`"axios":"^0.24.0",`   
`"core-js":"^3.6.5",`  
`"echarts":"^5.2.2",`    
`"element-ui":"^2.15.6",`   
`"less-loader": "^4.1.0",`   
`"lodash": "^4.17.21",`   
`"nprogress": "^0.2.0",`   
`"vue": "^2.6.11",`   
` "vue-quill-editor": "^3.0.6",`   
` "vue-router": "^3.2.0",`   
` "vue-table-with-tree-grid": "^0.2.4",`   
`"vuex": "^3.4.0"`   


## 通过以下命令来配置环境
```
npm install
cnpm install
```

### 运行
```
npm run serve
cnpm run serve
```

### 打包
```
npm run build
cnpm run build
```
